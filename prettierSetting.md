# .prettierrc setting

### Install `Prettier` extension in VS Code and create a file on root directory named by `.prettierrc` and paste the following setting

```json
{
  "tabWidth": 2,
  "semi": true,
  "singleQuote": true,
  "printWidth": 100,
  "bracketSpacing": true,
  "jsxBracketSameLine": true,
  "arrowParens": "always"
}
```

and in VS Code setting, search for `Format on save`, enable it and you are good to go üëç.
